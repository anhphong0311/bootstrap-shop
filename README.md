# 🛒 Bootstrap Shop

## 📖 Giới thiệu
Dự án này là một website bán hàng đơn giản được xây dựng bằng **HTML, CSS và Bootstrap 5**.  
Trang web hiển thị danh sách sản phẩm với giao diện hiện đại, dễ sử dụng và tương thích trên nhiều thiết bị.

---

## 🚀 Tính năng
- Hiển thị sản phẩm dạng card (ảnh, tên, mô tả, giá)
- Giao diện responsive (tự động co giãn trên mobile)
- Nút "Thêm vào giỏ"
- Banner và navbar đơn giản
- Sử dụng Bootstrap 5 (CDN)

---

## 🛠️ Công nghệ sử dụng
- HTML5
- CSS3
- Bootstrap 5

---

## 📂 Cấu trúc thư mục
bootstrap-shop/
│── index.html
│── images/

---

## ▶️ Cách chạy dự án
1. Tải project về máy
2. Mở bằng VS Code
3. Chuột phải `index.html` → **Open with Live Server**
